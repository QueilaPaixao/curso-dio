Listas Ordenadas e Listas não Ordenadas

Em HTML, as listas ordenadas e não ordenadas são usadas para organizar itens em uma sequência específica ou sem ordem particular.

Uma lista ordenada é usada quando os itens precisam ser exibidos em uma ordem específica, geralmente numerada. Os itens são marcados com números (por padrão) ou letras, dependendo da especificação do elemento se ol ou ul. Cada item da lista é definido pelo elemento li (item de lista), dentro da lista você pode fazer sua personalização como deixar em negrito, sublinhar ou deixar em itálico, também pode-se utilizar símbolos como square, disc ou circle, que são os marcadores padrão. Podendo ser enumeradas por:

  • Números
  • Letras
  • Símbolos

O que é Id?

Em HTML, id é um atributo usado para identificar de forma única um elemento na página. Cada id em um documento HTML deve ser exclusivo, ou seja, não pode haver mais de um elemento com o mesmo id dentro do mesmo documento. O atributo id é frequentemente usado em conjunto com CSS para estilizar elementos específicos ou com JavaScript para manipular o conteúdo desses elementos de forma dinâmica.

O que é Blockquote?

blockquote é uma tag HTML utilizada para representar um trecho de texto citado de outra fonte. Geralmente, os navegadores exibem o conteúdo de blockquote com uma formatação diferenciada, como uma margem indentada dos lados esquerdo e direito, para indicar visualmente que se trata de uma citação.

Estrutura básica no HTML

A estrutura básica de um documento HTML inclui as seguintes partes:

  • !DOCTYPE html: Declaração que define o tipo de documento e a versão do HTML sendo usada. Para HTML5, a declaração é simplesmente !DOCTYPE html.
  • html: Elemento raiz que contém todo o conteúdo da página.
  • head: Contém metadados sobre o documento, como título, codificação de caracteres, referências a arquivos de estilo e scripts, entre outros.
  • title: Define o título da página, exibido na barra de título do navegador ou na guia.
  • body: Contém o conteúdo principal do documento, como texto, imagens, links, etc.
